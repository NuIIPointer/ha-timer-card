# Timer Card

Create timers in your Lovelace dashboard that set any Home Assistant entity to a desired value when they expire. Timers run server-side and survive Home Assistant restarts.

- Multiple timers per card, multiple cards per dashboard
- Server-side execution (no automations required)
- Type-aware target value pickers (switch, light, number, climate, cover, fan, select, media_player)
- Tab-switcher between countdown duration and absolute time
- Quick-select favorites configurable per card
- German + English UI
